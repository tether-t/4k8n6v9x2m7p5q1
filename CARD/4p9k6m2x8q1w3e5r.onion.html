Getting Started

First Time Access

1 Download Tor Browser: Visit torproject.org and download the official Tor Browser. Never use regular browsers to access dark web sites.
2 Verify Onion Address: Our official address is 4k8n6v9x2m7p5q1r.onion. Always verify this address to avoid phishing sites.
3 Accept Terms: On first visit, you'll see our welcome page. Read and accept terms to access the main store.
Security Warning: Never bookmark our site or save passwords. Always type the onion address manually or use verified links from trusted sources.

Security Setup

Essential Security Measures

1 Tor Configuration: Use default Tor settings. Enable "Safest" security level in Tor Browser settings for maximum protection.
2 Disable JavaScript: Go to about:config and set javascript.enabled to false for enhanced security.
3 Use Tails OS: For maximum anonymity, use Tails operating system which routes all traffic through Tor automatically.

Bitcoin Security

1 Use Mixing Services: Mix your Bitcoin through services like CoinJoin before making purchases for enhanced privacy.
2 Fresh Addresses: Generate new Bitcoin addresses for each transaction. Never reuse addresses.

Troubleshooting

Common Issues & Solutions

Site Won't Load:
  • Try new Tor circuit (Ctrl+Shift+L)
  • Clear browser cache and cookies
  • Use Tor bridges if connection blocked
  • Verify onion address is correct
Payment Not Confirmed:
  • Check transaction on blockchain explorer
  • Ensure sufficient transaction fees
  • Wait for 3 confirmations minimum
  • Contact support if delayed over 2 hours
Product Not Working:
  • Verify card details entered correctly
  • Check if merchant accepts card type
  • Try different payment processor
  • Contact support for replacement
Emergency Contacts: If you suspect security breach or urgent issues, use our emergency PGP contact method available on the security page.
